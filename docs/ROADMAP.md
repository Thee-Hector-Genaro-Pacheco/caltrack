## Upcoming Milestones

### v0.9.0 — Predictive Calibration Intelligence

Add analytics for calibration trends, repeat failures, drift behavior, overdue risk, and high-risk instruments.

### v0.10.0 — Agent Memory

Add scoped memory for technician sessions, instrument history summaries, and recurring calibration patterns.

### v0.11.0 — RAG Documentation Assistant

Add document indexing, embeddings, and retrieval over manuals, SOPs, procedures, P&IDs, and loop drawings.

### v0.12.0 — MCP Integration

Expose CalTrack tools through Model Context Protocol-compatible interfaces.

### v0.13.0 — Android Field Technician App

Create a mobile technician workflow for field calibrations, work orders, QR scanning, and offline procedures.

### v1.0.0 — Production Portfolio Release

Stabilize architecture, polish UI, add deployment documentation, improve testing, and prepare live demo.
