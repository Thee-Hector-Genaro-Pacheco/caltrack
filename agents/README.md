# CalTrack Agent System

This folder defines the agentic architecture for CalTrack.

Each markdown file describes a specialized AI agent, its role, responsibilities, tool access, boundaries, and expected outputs.

The goal is to evolve CalTrack from an industrial calibration management platform into an agentic industrial operations assistant.

Agents are human-in-the-loop by design. They may analyze, recommend, summarize, and draft actions, but they must not automatically approve, reject, delete, or modify compliance records without user confirmation.
